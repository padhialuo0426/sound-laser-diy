## 简介
SoundLaserDIY 是一个基于超声波参量阵(Parametric Array)原理的纯硬件开源项目。它能够将音频信号调制到超声波频率，像激光束一样定向投射声音，实现“只有特定位置能听到”的奇妙声学体验。 该项目包含完整的调制电路设计与超声波换能器阵列驱动方案，适合电子爱好者和DIY爱好者进行学习和实践。

## 原理

- 项目利用了空气的非线性声学效应（Nonlinear Acoustics）。

- **载波生成**：由 NE555 (U1) 产生中心频率约为 40kHz 的高频方波载波 。

- **PWM 调制**：音频信号与载波结合，转化为脉冲宽度调制（PWM）信号。

- **功率放大**：通过 TC4427 (U3-U12) 高速驱动芯片，将微弱信号放大至足以驱动换能器阵列的功率 。

- **自解调**：高功率超声波在空气中传播时，由于空气的非线性特性，信号会发生自解调（Self-demodulation），从而在空气中还原出可听的音频信号。
