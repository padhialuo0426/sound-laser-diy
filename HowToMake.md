# 制作方法
本文档介绍如何制作 SoundLaserDIY 声束发射器，包括所需材料、工具和步骤。

# 所需材料
- 超声波发射探头（建议使用 **40kHz** 的探头，具体参数在 **超声波探头的参数.png** 图片中有详细介绍），数量根据设计需求决定，我制作的样品使用了100个探头，排列成10x10的矩阵；
- 洞洞板，用于安装和连接超声波发射探头；
- 电源适配器（使用 **12V** 直流电源，PCB上面有一个DC插孔用于连接电源适配器）；
- 公对公3.5mm音频线，用于连接音频源设备和PCB板上的音频输入插孔；
- 带有3.5mm音频插孔的音频源设备（如MP3、电脑等）；
- 其余电子元件和PCB设计在文件夹 **PCB Design** 中有详细说明；
  
# 所需工具
- 烙铁和焊锡丝，用于焊接电子元件；
- 没了
  
# 制作步骤
- 参考 **Driver Board PCB Design** 文件夹中的电路原理图和PCB布局设计，准备好所有电子元件和PCB板；
- 将电子元件按照电路原理图焊接到PCB板上，确保焊接牢固且无短路；
- 将超声波发射探头焊接到洞洞板上，可以参考我提供的样品照片进行排列和连接，不要搞反超声波探头的极性；
- 本人使用了10x10的矩阵排列方式，将100个探头均匀分布在洞洞板上，设计的 PCB 有 20 路输出，每路连接 5 个探头并联，具体连接方式可以参考样品照片；
- 然后将所有的探头共地连接到 PCB 板的 GND 引脚上，PCB 板上有标注 GND 的焊盘；
- 连接电源适配器到 PCB 板的 DC 插孔，确保电源适配器输出为 12V 直流电压；
- 使用公对公3.5mm音频线将音频源设备连接到 PCB 板上的音频输入插孔；
- 打开音频源设备，播放音乐或其他音频内容，调整音量到合适水平；
- 观察和测试声束发射效果，可以在特定位置听到定向的声音效果。

# 注意事项
- 确保所有焊接点牢固且无短路，以避免电路故障；
- 使用合适的电源适配器，避免过高或过低的电压；
- 在测试过程中，注意保护听力，避免长时间暴露在高音量环境中；
- 千万不要对着有降噪功能的耳机或助听器使用，以免损坏设备或伤害听力。
